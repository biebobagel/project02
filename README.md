# Project02: Automated Turntable and Lighting Setup Tool

## Summary
### This is an automated turntable and lighting setup tool. This tool will allow a use to select an object and automatically generate a complete lookdev setup. This tool button when pushed, will create a camera, a three-point lighting system, and animate the object in a turntable rotation. The result will be a visually appealing rendered animation.
